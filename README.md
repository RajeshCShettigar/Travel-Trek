# TravelTrek :
## Description :
Traveltrek is tour planner application built using MERN stack.

## Technologies Used :
- ReactJs
- NodeJs
- ExpressJs
- MongoDB
- Tailwind CSS
- Cloudinary

## Installation :
1. Clone the repository
2. Install dependencies
```
npm install
```
3. Run the application
```
npm run dev
```
4. Open http://localhost:5173 to view it in the browser.
